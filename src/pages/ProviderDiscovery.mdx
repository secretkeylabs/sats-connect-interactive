# getProviders & getProviderById

Use <code>getProviders()</code> to inspect the wallet providers currently available in the browser, and <code>getProviderById()</code> when you want the raw provider object for one specific wallet.

<Callout type="info">
  These helpers do not require account permissions. They only inspect provider
  metadata already injected into the page.
</Callout>
