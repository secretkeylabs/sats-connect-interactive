# wallet_getNetwork

Use the <code>wallet_getNetwork</code> method to retrieve the currently
selected network configuration for Bitcoin, Stacks, and Spark.

<Callout type="info">
  Your app must first be connected to the wallet. No parameters are required.
</Callout>

## Try it

<GetNetworkExample />

## Response

Returns an object with the network names for each chain:
