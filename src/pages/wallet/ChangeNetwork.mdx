# wallet_changeNetwork

Use the <code>wallet_changeNetwork</code> method to switch the active Bitcoin network in the connected wallet.

<Callout type="info">
  In <code>sats-connect@4.2.1</code>, this method accepts a Bitcoin network
  name. If your wallet has assigned ids to custom networks, use
  <code>wallet_changeNetworkById</code> for id-based switching.
</Callout>

## Parameters

<ParamTable
  params={[
    {
      name: "name",
      type: "BitcoinNetworkType",

  ]}
