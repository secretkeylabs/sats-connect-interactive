# wallet_getAccount

Use the <code>wallet_getAccount</code> method to retrieve account information for the currently connected wallet. This includes the account ID, wallet type, and all associated addresses.

<Callout type="info">
  This method requires the user to have previously connected their wallet to
  your app. If the app doesn't have permissions, you'll receive an
  <code>ACCESS_DENIED</code> error.
</Callout>
