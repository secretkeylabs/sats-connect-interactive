# wallet_getWalletType

Use the <code>wallet_getWalletType</code> method to determine the type of wallet the user is using — whether it's a software wallet, Ledger hardware wallet, or Keystone hardware wallet.

<Callout type="info">
  Your app must first be connected to the wallet. No parameters are required.
</Callout>

## Try it

<GetWalletTypeExample />

## Response

Returns one of: <code>"software"</code>, <code>"ledger"</code>, or <code>"keystone"</code>.
