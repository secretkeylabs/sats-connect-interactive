<H1>wallet_requestPermissions</H1>

<P>
  Use the <code>wallet_requestPermissions</code> method to explicitly request
  permissions from the user's wallet. This is an alternative to{" "}
  <code>wallet_connect</code> that focuses solely on obtaining permissions
  without returning address data.
</P>

<Callout type="info">
  In most cases, using <code>wallet_connect</code> is preferred as it combines
  permission granting with address retrieval in a single step. Use{" "}
  <code>wallet_requestPermissions</code> when you need fine-grained control over
  the permission flow.
</Callout>
