# wallet_requestPermissions

Use the <code>wallet_requestPermissions</code> method to explicitly request wallet permissions without also returning account data.

<Callout type="info">
  In most cases, using <code>wallet_connect</code> is preferred as it combines
  permission granting with address retrieval in a single step. Use
  <code>wallet_requestPermissions</code> when you need fine-grained control over
  the permission flow.
</Callout>
