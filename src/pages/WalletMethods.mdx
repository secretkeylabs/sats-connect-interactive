# Wallet Methods

The docs in this site cover two related method families:

- standard request methods for Bitcoin, Stacks, Spark, Runes, and Ordinals
- <code>wallet_*</code> methods for connection, permissions, wallet inspection,
  and network management

<Callout type="info">
  Use [request methods](/wallet/request-methods) for the transport and response
  model. Use [Xverse custom methods](/wallet/custom-methods) for the
  wallet-specific extensions currently documented in this repo.
</Callout>
