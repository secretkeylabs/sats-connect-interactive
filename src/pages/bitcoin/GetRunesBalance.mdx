# runes_getBalance

Use the <code>runes_getBalance</code> method to retrieve the Runes token
balances held by the connected wallet.

<Callout type="info">
  Your app must first be connected to the wallet. No parameters are required.
</Callout>

## Try it

<GetRunesBalanceExample />

## Response

Returns an object with a <code>balances</code> array. Each balance includes:
