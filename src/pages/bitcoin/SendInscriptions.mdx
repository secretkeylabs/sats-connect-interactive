# ord_sendInscriptions

Use the <code>ord_sendInscriptions</code> method to send ordinal inscriptions
from the connected wallet to a recipient address.

## Parameters

<ParamTable
  params={[
    {
      name: "transfers",
      type: "Transfer[]",
      required: true,
      description:
        "Array of transfer objects, each with an address and inscriptionId.",
    },
  ]}
/>

Each transfer object contains:

<ParamTable
  params={[
    {
      name: "address",
      type: "string",
      required: true,
      description: "The recipient Bitcoin address.",
    },
    {
      name: "inscriptionId",
      type: "string",
      required: true,
      description: "The ID of the inscription to send.",
    },
  ]}
/>
