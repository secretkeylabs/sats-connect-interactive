# runes_etch

Use the <code>runes_etch</code> method to create (etch) a new Rune on the Bitcoin blockchain. This defines the Rune's properties including name, symbol, and minting terms.

## Parameters

<ParamTable
  params={[
    {
      name: "runeName",
      type: "string",
      required: true,
      description: "The name of the Rune to create.",
    },
    {
      name: "isMintable",
      type: "boolean",
      required: true,
      description: "Whether the Rune can be minted after creation.",
    },
    {
      name: "destinationAddress",
      type: "string",
      required: true,
      description: "The ordinals address to receive the etched Rune.",
    },
    {
      name: "refundAddress",
      type: "string",
      required: true,
      description: "The payment address for refunds.",
    },
    {
      name: "feeRate",
      type: "number",
      required: true,
      description: "The fee rate in sats/vB.",
    },
    {
      name: "symbol",
      type: "string",
      required: false,
      description: "A symbol for the Rune (e.g. 'R').",
    },
    {
      name: "terms",
      type: "{ amount: string, cap: string }",
      required: false,
      description: "Mint terms: amount per mint and total supply cap.",
    },
  ]}
/>
