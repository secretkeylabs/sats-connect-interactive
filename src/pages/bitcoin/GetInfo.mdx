# getInfo

Use the <code>getInfo</code> method to retrieve information about the
connected wallet, such as its version, platform, and supported methods.

<Callout type="info">
  Your app must first be connected to the wallet. No parameters are required.
</Callout>

## Try it

<GetInfoExample />

## Response

The <code>getInfo</code> method returns an object with wallet metadata:

<ParamTable
  params={[
    {
      name: "version",

    {
      name: "methods",

    {
      name: "supports",
