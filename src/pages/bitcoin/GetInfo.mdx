# getInfo

Use the <code>getInfo</code> method to retrieve metadata about the selected wallet provider, such as its version, platform, and supported request methods.

<Callout type="info">
  No prior account permissions are required. If no provider has been selected
  yet, calling <code>Wallet.request("getInfo", null)</code> may first prompt the
  user to choose a compatible wallet.
</Callout>

## Response

The <code>getInfo</code> method returns an object with wallet metadata:

<ParamTable
  params={[
    {
      name: "version",
      type: "string",
      required: true,
      description: "Version of the wallet.",
    },
    {
      name: "platform",
      type: '"web" | "mobile"',
      required: false,
      description:
        "Platform the wallet is running on, when provided by the wallet.",

    {
      name: "methods",
      type: "string[]",

      description: "Methods supported by the wallet.",
    },
    {
      name: "supports",
      type: "string[]",
      required: true,
      description: "List of supported standards or capability identifiers.",
    },
  ]}
