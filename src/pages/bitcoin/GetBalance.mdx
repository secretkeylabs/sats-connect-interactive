# getBalance

Use the <code>getBalance</code> method to retrieve your user's Bitcoin balance
— the amount of BTC that their connected wallet's Bitcoin payment address
holds.

<Callout type="info">
  Your app must first be connected to the wallet and have account read
  permissions. No query parameters are required.
</Callout>

## Try it

<GetBalanceExample />

## Response

The <code>getBalance</code> method returns an object representing the
connected wallet's payment address BTC holdings:

<ParamTable
  params={[
    {
      name: "confirmed",
      type: "string",
      required: true,
      description: "Confirmed balance in satoshis.",
    },
    {
      name: "unconfirmed",
      type: "string",
      required: true,
      description: "Unconfirmed balance in satoshis.",
    },
    {
      name: "total",
      type: "string",
      required: true,
      description: "Total balance (confirmed + unconfirmed) in satoshis.",
    },
  ]}
/>

<Callout type="tip">
  This method does not trigger any wallet prompt or signing request. It simply
  reads the balance from the connected account.
</Callout>
