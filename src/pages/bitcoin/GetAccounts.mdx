# getAccounts

Use the <code>getAccounts</code> method to request wallet account details for
specific purposes. Similar to <code>getAddresses</code> but returns additional
wallet type information.

## Parameters

<ParamTable
  params={[
    {
      name: "purposes",
      type: "AddressPurpose[]",
      required: true,
      description:
        "The purposes for which to retrieve accounts (Payment, Ordinals, Stacks).",
    },
    {
      name: "message",
      type: "string",
      required: false,
      description: "A message to display to the user in the request prompt.",
    },
  ]}
/>

## Try it

<GetAccountsExample />

## Response

Returns an array of account objects, each containing:

<ParamTable
  params={[
    {
      name: "address",
      type: "string",
      required: true,
      description: "The wallet address.",
    },
    {
      name: "publicKey",
      type: "string",
      required: true,
      description: "The public key associated with the address.",
    },
    {
      name: "purpose",
      type: "AddressPurpose",
      required: true,
      description: "The purpose of the account.",
    },
    {
      name: "walletType",
      type: '"software" | "ledger" | "keystone"',
      required: true,
      description: "The wallet type for this account.",
    },
  ]}
/>
