# getAddresses

Use the <code>getAddresses</code> method to request wallet addresses for specific purposes like payment, ordinals, or Stacks. The wallet will prompt the user to share their addresses.

## Parameters

<ParamTable
  params={[
    {
      name: "purposes",
      type: "AddressPurpose[]",
      required: true,
      description:
        "The purposes for which to generate addresses (Payment, Ordinals, Stacks).",
    },
    {
      name: "message",
      type: "string",
      required: false,
      description: "A message to display to the user in the request prompt.",
    },
  ]}
/>
