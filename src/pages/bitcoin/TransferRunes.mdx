# runes_transfer

Use the <code>runes_transfer</code> method to transfer Runes tokens to one or more recipients.

## Parameters

<ParamTable
  params={[
    {
      name: "recipients",
      type: "Recipient[]",
      required: true,
      description: "Array of recipient objects.",
    },
  ]}
/>

Each recipient object contains:

<ParamTable
  params={[
    {
      name: "runeName",
      type: "string",
      required: true,
      description: "The name of the Rune to transfer.",
    },
    {
      name: "amount",
      type: "string",
      required: true,
      description: "The amount to transfer.",
    },
    {
      name: "address",
      type: "string",
      required: true,
      description: "The recipient Bitcoin address.",
    },
  ]}
/>
