# runes_mint

Use the <code>runes_mint</code> method to mint Runes tokens. This creates a
funding transaction to mint a specified Rune.

## Parameters

<ParamTable
  params={[
    {
      name: "runeName",
      type: "string",
      required: true,
      description: "The name of the Rune to mint.",
    },
    {
      name: "destinationAddress",
      type: "string",
      required: true,
      description: "The ordinals address to receive the minted Runes.",
    },
    {
      name: "refundAddress",
      type: "string",
      required: true,
      description: "The payment address for refunds.",
    },
    {
      name: "feeRate",
      type: "number",
      required: true,
      description: "The fee rate in sats/vB.",
    },
    {
      name: "repeats",
      type: "number",
      required: true,
      description: "Number of mint transactions.",
    },
  ]}
/>
