# spark_getBalance

Use the <code>spark_getBalance</code> method to retrieve the user's Spark
balance. Spark is a Bitcoin Layer 2 that enables instant, low-fee BTC
transfers.

<Callout type="info">
  Your app must be connected to the wallet with Spark address permissions before
  calling this method.
</Callout>
