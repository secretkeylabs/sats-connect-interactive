# spark_getAddresses

Use the <code>spark_getAddresses</code> method to retrieve the Spark layer addresses from the connected wallet.

<Callout type="info">
  Your app must first be connected to the wallet. No parameters are required.
</Callout>

## Try it

<SparkGetAddressesExample />

## Response

Returns an object with an <code>addresses</code> array containing Spark address details:

<ParamTable
  params={[
    {
      name: "address",
      type: "string",
      required: true,
      description: "The Spark address.",
    },
    {
      name: "publicKey",
      type: "string",
      required: true,
      description: "The public key associated with the address.",
    },
    {
      name: "purpose",
      type: "AddressPurpose",
      required: true,
      description: "The purpose of the address.",
    },
  ]}
/>
