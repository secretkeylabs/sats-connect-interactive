# spark_signMessage

Use the <code>spark_signMessage</code> method to sign a message with the user's Spark key. This can be used for authentication or identity verification on Spark.

## Parameters

<ParamTable
  params={[
    {
      name: "message",
      type: "string",
      required: true,
      description: "The message to sign.",
    },
    {
      name: "publicKey",
      type: "string",
      required: true,
      description: "The user's Spark public key.",
    },
    {
      name: "protocol",
      type: "'ECDSA'",
      required: false,
      description: "Signing protocol. Currently only ECDSA is supported.",
    },
  ]}
/>
