# stx_callContract

Use the <code>stx_callContract</code> method to call a function on a deployed Stacks smart contract. The wallet will prompt the user to sign and broadcast the transaction.

## Parameters

<ParamTable
  params={[
    {
      name: "contract",
      type: "string",
      required: true,
      description: 'The contract principal (e.g. "SP000...78.contract-name").',
    },
    {
      name: "functionName",
      type: "string",
      required: true,
      description: "The name of the function to call.",
    },
    {
      name: "arguments",
      type: "string[]",
      required: false,
      description: "Hex-encoded Clarity values as function arguments.",
    },
  ]}
/>

<Callout type="tip">
  To convert Clarity values to hex, use the <code>cvToHex</code> helper from the
  <code>@stacks/transactions</code> package.
</Callout>
