<H1>stx_signTransaction</H1>

<P>
  Use the <code>stx_signTransaction</code> method to sign a Stacks transaction.
  The transaction should be provided as a hex-encoded string. Optionally, the
  wallet can broadcast it after signing.
</P>

<H2>Parameters</H2>

<ParamTable
  params={[
    {

      description: "The unsigned Stacks transaction as a hex-encoded string.",
    },
    {
      name: "pubkey",
      type: "string",
      required: false,
      description:
        "The public key to sign with. The wallet may use any key when not provided.",
    },
    {
      name: "broadcast",
      type: "boolean",
      required: false,
      description: "Whether to broadcast after signing. Defaults to true.",
    },
  ]}
/>

  <code>@stacks/transactions</code> package with functions like{" "}

</Callout>
