# stx_getAddresses

Use the <code>stx_getAddresses</code> method to retrieve the Stacks addresses
from the connected wallet.

<Callout type="info">
  Your app must first be connected to the wallet. No parameters are required.
</Callout>

## Try it

<StxGetAddressesExample />

## Response

Returns an object with an <code>addresses</code> array containing Stacks
address details:

<ParamTable
  params={[
    {
      name: "address",
      type: "string",
      required: true,
      description: "The Stacks address.",
    },
    {
      name: "publicKey",
      type: "string",
      required: true,
      description: "The public key associated with the address.",
    },
  ]}
/>
