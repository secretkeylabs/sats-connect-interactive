# stx_getAccounts

Use the <code>stx_getAccounts</code> method to retrieve the Stacks account
details from the connected wallet, including addresses and Gaia hub
information.

<Callout type="info">
  Your app must first be connected to the wallet. No parameters are required.
</Callout>

## Try it

<StxGetAccountsExample />

## Response

Returns an object with an <code>addresses</code> array and network
information:
