# stx_transferStx

Use the <code>stx_transferStx</code> method to request a transfer of STX
tokens from the user's Stacks wallet to a recipient address.

## Parameters

<ParamTable
  params={[
    {
      name: "recipient",
      type: "string",
      required: true,
      description: "The Stacks address to send STX to.",
    },
    {
      name: "amount",
      type: "number",
      required: true,
      description: "Amount in micro-STX (1 STX = 1,000,000 uSTX).",
    },
    {
      name: "memo",
      type: "string",
      required: false,
      description: "Optional memo to attach to the transaction.",
    },
  ]}
/>
