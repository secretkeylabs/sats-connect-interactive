# stx_deployContract

Use the <code>stx_deployContract</code> method to deploy a Clarity smart contract to the Stacks blockchain. The wallet will prompt the user to sign and broadcast the deployment transaction.

## Parameters

<ParamTable
  params={[
    {
      name: "name",
      type: "string",
      required: true,
      description: "The name for the contract.",
    },
    {
      name: "clarityCode",
      type: "string",
      required: true,
      description: "The Clarity source code of the contract.",
    },
    {
      name: "clarityVersion",
      type: "number",
      required: false,
      description: "The version of Clarity to use.",
    },
  ]}
/>
