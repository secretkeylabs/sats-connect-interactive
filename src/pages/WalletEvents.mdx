# Xverse Wallet Events

Use <code>Wallet.addListener()</code> to keep your UI in sync with account, network, and disconnect changes after the user has connected.

<Callout type="info">
  The shipped event names in <code>sats-connect@4.2.1</code> are
  <code>accountChange</code>, <code>networkChange</code>, and
  <code>disconnect</code>.
</Callout>
