<H1>Connect Wallet</H1>

<P>
  An app can connect to the user's wallet using <code>wallet_connect</code>.
  This method grants read permissions on the account selected by the user and
  returns commonly used data about the account, such as addresses and current
  network.
</P>

<Callout type="info">
  If the user has already granted permissions, no connection request popup will
  appear, ensuring a smooth user experience.
</Callout>
