# Disconnect Wallet

The current SDK exposes two request-level ways to end a session:

- <code>request("wallet_disconnect", null)</code>
- <code>request("wallet_renouncePermissions", null)</code>

The convenience helper <code>Wallet.disconnect()</code> currently wraps

<code>wallet_renouncePermissions</code>, clears the stored provider id, and
removes the default provider selection.
